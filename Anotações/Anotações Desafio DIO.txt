[GIT](https://git-scm.com/)
O GIT BASH é um terminal extendido para para otimizar o uso do GIT

## Comandos

git clone + site (add repositório)
git add *
git commit -m "    "
git push origin main (enviar commit para nuvem/git hub)
git status
ls
dir: lista todas as pastas (diretórios)
cls  limpa a tela
cd  + nome = navega para pasta
cd .. volta 
TAB autocompletar 
mkdir + nome = cria uma pasta
del + nome do arquivo = deleta o arquivo
echo + texto = echo hello
echo hello > hello.txt  (cria arquivo)
git pull (trazer arquivos da nuvem/git hub)
